# Viewing the Presentation (Non‑Technical)

The easiest way is to double‑click the helper file below. It starts a tiny local viewer and opens the presentation in your browser.

## One‑click viewer (Mac)

1. Double‑click `view_presentation.command`.
2. If macOS warns you, right‑click it → **Open**.

That’s it.

## One‑click viewer (Windows)

1. Double‑click `view_presentation.bat`.

That’s it.
